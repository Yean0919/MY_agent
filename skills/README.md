# Skills 目录

本目录包含所有 Skill 定义。

## Skill 结构

每个 Skill 是一个目录，包含：

```
skill_name/
├── SKILL.md          # Skill 定义文件
└── scripts/          # 脚本目录
    └── script.py     # 可执行脚本
```

## SKILL.md 格式

```markdown
# 名称
skill_name

# 描述
Skill 的描述

# 版本
1.0.0

# 使用方法
如何使用这个 Skill

# 参数
- `--param`: 参数描述
```
