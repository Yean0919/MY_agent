# Terminal CodingAgent 文档

本目录包含项目的所有文档。

## 文档清单

| 文档 | 用途 |
|------|------|
| `01_产品需求文档.md` | PRD：功能边界、非功能指标、验收标准 |
| `02_系统架构文档.md` | 技术架构：分层、模块、数据流、关键决策 |
| `03_项目开发文档.md` | 实施约定：环境、目录、编码规范、构建与测试 |
| `04_核心模块设计.md` | 模块设计：5 大差异化模块的接口、流程、可执行示例 |
| `05_测试与运维手册.md` | 质量门禁：测试金字塔、CI/CD、部署与运维 |

## 阅读顺序

按编号顺序阅读：
1. 先读 `01_产品需求文档.md` 了解项目目标
2. 再读 `02_系统架构文档.md` 了解技术架构
3. 然后读 `03_项目开发文档.md` 了解开发规范
4. 接着读 `04_核心模块设计.md` 了解模块设计
5. 最后读 `05_测试与运维手册.md` 了解测试与部署
